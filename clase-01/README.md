# Clase 01 - Introducción a Python

## Modos de ejecución d elos scripts

1. Una vez instaladas la extensiones recomendas.

Hacer clic sobre el botón play que aparece a la derecha arriba de los archivos .py

2. Abrir la terminal y ejecutar los siguientes comandos

```sh
py <nombre-script.py>
py main.py
python main.py
```
